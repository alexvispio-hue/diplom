# Структура описания проекта в дипломе

## 1. Постановка задачи

Описать проблему распознавания рукописного текста, область применения и ограничения существующих OCR-систем.

## 2. Архитектура системы

Показать модульную схему:

- пользовательский интерфейс;
- FastAPI backend;
- сервис распознавания;
- предобработка изображений;
- локальная нейросетевая модель;
- хранилище истории;
- модуль экспериментальной оценки.

## 3. Реализация

Описать основные модули проекта:

- `ImagePreprocessor`;
- `TrOCRRecognizer`;
- `RecognitionService`;
- `EvaluationService`;
- SQLite-модель истории.

## 4. Экспериментальная оценка

Описать тестовый набор, эталонные тексты и метрики:

- CER;
- WER;
- время обработки;
- анализ типичных ошибок.

## 5. Выводы

Зафиксировать, что приложение выполняет локальное распознавание без внешнего OCR API, а архитектура допускает замену модели и дообучение под русский рукописный текст.
