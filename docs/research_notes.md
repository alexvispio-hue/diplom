# Заметки по дипломной части

## Основная тема

Разработка системы распознавания рукописного текста на изображениях с использованием методов глубокого обучения.

## Выбранная модель

Для рабочей русскоязычной версии используется `kazars24/trocr-base-handwritten-ru`. Это модель архитектуры TrOCR, выполняющая локальное распознавание коротких русских рукописных фрагментов.

## Данные

В проект локально подключен `Cyrillic Handwriting Dataset`:

- изображения строковых или коротких текстовых фрагментов;
- русские эталонные подписи в `train.tsv` и `test.tsv`;
- `72 286` обучающих и `1 544` тестовых изображения.

## Метрики

Для экспериментальной оценки используются:

- Character Error Rate;
- Word Error Rate;
- расстояние Левенштейна;
- время обработки изображения.

CER:

```text
CER = edit_distance(reference_chars, hypothesis_chars) / len(reference_chars)
```

WER:

```text
WER = edit_distance(reference_words, hypothesis_words) / len(reference_words)
```

## Эксперименты

- baseline-оценка русской модели на тестовом split;
- сравнение результата с предобработкой и без нее;
- fine-tuning модели на train split;
- сравнение baseline и fine-tuned checkpoint;
- последующее тестирование на фотографиях реальных записей.
