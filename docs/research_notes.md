# Заметки по дипломной части

## Основная тема

Разработка системы распознавания рукописного текста на изображениях с использованием методов глубокого обучения.

## Базовая модель

Для первой версии используется TrOCR. Это transformer-based OCR-модель, которая объединяет визуальный encoder и текстовый decoder.

## Метрики

Для экспериментальной оценки используются:

- Accuracy;
- Character Error Rate;
- Word Error Rate;
- расстояние Левенштейна.

CER:

```text
CER = edit_distance(reference_chars, hypothesis_chars) / len(reference_chars)
```

WER:

```text
WER = edit_distance(reference_words, hypothesis_words) / len(reference_words)
```

Где `reference` — эталонный текст, а `hypothesis` — текст, распознанный моделью.

## Возможные эксперименты

- сравнение качества на исходных и предобработанных изображениях;
- сравнение TrOCR и CRNN;
- анализ ошибок на разных стилях почерка;
- оценка времени обработки одного изображения;
- проверка устойчивости к шумам, наклону, плохому освещению.
