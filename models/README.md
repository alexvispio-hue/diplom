# Models

По умолчанию приложение использует локальный запуск русскоязычной модели `kazars24/trocr-base-handwritten-ru`.

Она основана на TrOCR и дообучена на `Cyrillic Handwriting Dataset`, который содержит русские рукописные фрагменты.

При первом запуске веса модели загружаются библиотекой Hugging Face Transformers в локальный кэш. Это не внешний OCR API: распознавание выполняется внутри приложения через PyTorch.
